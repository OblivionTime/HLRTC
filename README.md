# HLRTC(基于webrtc实现的demo)

基于webrtc实现的数据通道(datachannel)实现p2p私聊和群聊,实现多用户屏幕共享和多用户视频,后端纯nodejs实现,简单实用,适合新手练习webrtc

## ❗本地运行

```js
//获取到项目
git clone https://github.com/OblivionTime/HLRTC.git
//运行ui
cd ui
yarn
//本地测试
yarn serve
//打包
yarn build


//运行服务端
cd server
yarn


```
